Solicitar crédito

Crédito vía nómina para trabajadores elegibles.

Esta información está dirigida a personas trabajadoras que pertenecen a una entidad con convenio vigente y desean conocer el proceso general para solicitar un crédito vía nómina con GPF.

Proceso general

El proceso puede variar según convenio, documentación, validaciones y condiciones vigentes.

01

Contacto

Comparte tus datos para identificar el tipo de solicitud y la entidad a la que perteneces.

02

Validación

Se revisa si existe convenio vigente, perfil elegible y documentación básica.

03

Análisis

Se analiza capacidad de pago, condiciones del producto y validaciones necesarias.

04

Formalización

Si aplica, se informan condiciones y se completa el proceso correspondiente.

05

Dispersión

Si la solicitud es autorizada y formalizada, se realiza la dispersión conforme al proceso aplicable.

Qué es

Una solución de crédito vinculada al empleo y al descuento vía nómina.

El crédito vía nómina permite que el pago se realice mediante descuentos programados al salario, siempre que la entidad, el trabajador y la operación cumplan las reglas del convenio.

Qué debes considerar

Solicitar información no significa aprobación automática. El proceso requiere revisión y validación antes de definir condiciones finales.

  • El producto está sujeto a convenio vigente con la entidad empleadora.
  • El monto y plazo dependen del análisis correspondiente.
  • Las condiciones, pagos, plazos, CAT y comisiones deben informarse antes de contratar cuando aplique.

Importante: GPF no ofrece aprobación garantizada. Toda solicitud está sujeta a elegibilidad, análisis, documentación, capacidad de pago, convenio vigente y validación correspondiente.
